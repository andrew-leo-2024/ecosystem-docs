# Documentation Roadmap

With v1.0 launched, here's what's coming next for the AINEFF Ecosystem documentation.

<!-- truncate -->

## Planned Enhancements

### Interactive Features
- **Search** — Full-text local search across all 124 pages
- **Tags** — Tag-based navigation across altitude levels, content types, and domains
- **PWA** — Offline access for field reference

### Content Expansion
- API reference documentation for protocol interfaces
- Integration guides for AINE enterprise deployments
- Operator training modules linked from SOPs
- Case study templates for each market wedge

### Governance Tracking
- Document lifecycle metadata (status, owner, review dates)
- Governance dashboard showing documentation coverage
- Automated review reminders for constitutional documents

## Contributing

All documentation lives in the `docs/` directory. Each page follows the standard frontmatter format with sidebar position, title, and description. Mermaid diagrams are written inline using fenced code blocks.
